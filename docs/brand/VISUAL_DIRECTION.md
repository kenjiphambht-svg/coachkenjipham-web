# Visual Direction

The visual direction should feel premium, quiet, warm, and refined.

## Core Direction

- Premium, quiet, warm, refined
- Dark warm background
- Cream text
- Muted gold accents
- Spacious layout
- Serif headings, clean readable body
- Mỗi màn hình một ý
- Khoảng trống là vật liệu chính
- Chuyển động chỉ ở mức hơi thở

## Premium Screen Principles

- Mỗi màn hình một ý: một section should carry one message and, when needed, one clear action.
- Khoảng trống là vật liệu chính: prefer fewer elements, more vertical room, and calmer hierarchy.
- Chuyển động chỉ ở mức hơi thở: keep movement subtle, slow, and respectful of `prefers-reduced-motion`.

## Avoid

- Neon SaaS look
- Generic startup template
- Spiritual guru aesthetic
- Loud gradients
- Overly decorative motion
- Dense service-menu layouts

## Motion

Motion should be subtle:

- mist
- breath
- warm light
- slow reveal
- restrained hover states

Do not add animation libraries without explicit approval. Use existing motion capability first.

## Mobile

Mobile must be designed intentionally, not just shrunk from desktop.

Check:

- hierarchy
- spacing
- CTA visibility
- navigation behavior
- readable line length
- motion restraint
