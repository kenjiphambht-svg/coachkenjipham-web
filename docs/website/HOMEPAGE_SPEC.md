# Homepage Specification

This document defines the future homepage structure. Do not implement this rebuild until explicitly asked.

## Homepage Role

The homepage is a 3-flow routing hall:

- For yourself
- For your child
- For understanding Kenji / Essence / the AI system

The homepage should be simple on the outside and deep on the inside. It should help visitors choose the right door without turning the site into a crowded service menu.
