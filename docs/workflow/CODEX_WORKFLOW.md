# Codex Workflow

Use this process for Codex tasks in this repository.

## Process

1. Create or work on a branch per task.
2. Read `AGENTS.md` first.
3. Read relevant docs before editing.
4. Keep task scope small.
5. Do not install dependencies unless approved.
6. Do not touch unrelated routes.
7. Run build/lint if possible.
8. Report changed files.
9. Report verification results.
10. Report unresolved risks.

## Guardrails

- Do not modify `/ai-startup` unless explicitly asked.
- Do not modify `/kidbook` unless explicitly asked.
- Do not modify payment pages unless explicitly asked.
- Do not modify private publication routes unless explicitly asked.
- Do not change `package.json` unless explicitly approved.
- Do not add dependencies unless explicitly approved.
