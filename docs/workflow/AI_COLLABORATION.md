# AI Collaboration

This repository may be touched by multiple AI assistants, but GitHub remains the source of truth and only one AI should edit the same source files at a time.

## Roles

- Kenji: founder / creative director / final decision maker
- ChatGPT: strategy architect / brief writer / QA / copy and safety reviewer
- Codex: primary repo implementation agent
- Claude Cowork: research, content drafts, checklist, operational assistant
- Claude Code: local code assistant, skills, review, preview loop
- GitHub: source of truth
- Vercel: preview and current deployment platform
- Cloudflare: later infrastructure target after website stabilizes

## Collaboration Rules

- Only one AI should edit the same source files at a time.
- Use branches for scoped work.
- Keep changes small and reviewable.
- Read `AGENTS.md` before editing.
- Read relevant docs before editing affected routes or brand copy.
- Do not merge or deploy automatically.
- Human review is required before merge.
