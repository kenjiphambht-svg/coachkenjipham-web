# QA Checklist

Use this checklist before handing work back.

- [ ] Build passes
- [ ] Lint/typecheck checked if available
- [ ] No unrelated routes changed
- [ ] Mobile layout checked
- [ ] Desktop layout checked
- [ ] CTA links checked
- [ ] SEO title/description checked
- [ ] No forbidden language
- [ ] Child language safety checked
- [ ] Payment/private pages untouched
- [ ] No new dependency added
- [ ] Changed files listed
